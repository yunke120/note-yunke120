表格

属性

  • table 表格
  • tr 行 每一行可以包含一个或多个td或th
  • td 表示表格中的标准单元格
  • th 表头单元格

    • table 属性

  • border 边框
  • width 宽度 [像素， 百分比] 参考的是上一级元素的宽度，如果未设置则参考屏幕宽度
  • height 高度
  • align 表格对其方式

    • tr属性

  • align 横向对齐 [left center right]
  • valign 纵向对齐 [top center bottom]
  • bgcolor 设置行的背景颜色

    • td

  • colspan 纵向合并单元格
  • rowspan 横向合并单元格

    • example

    class name age gender
    01 java 18 man
    02 html 16 woman

    纵向合并

    class name age gender
    01 java 18 man
    html 16 woman

    横向合并

    class+name age gender
    01 java 18 man
    02 html 16 woman
